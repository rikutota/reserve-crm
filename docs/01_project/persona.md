# 顧客ペルソナ

## 1. 店舗情報

- 業種：美容サロン
- 店舗数：1店舗
- スタッフ数：3名
- 月間予約数：約200件
- 主な予約経路：電話、LINE、Instagram DM、店頭
- 主な顧客層：20代〜50代の女性

## 2. 登場人物

### オーナー

店舗の経営者。
予約状況、売上、スタッフ別の実績を把握したい。

主な関心ごと：

- 売上を把握したい
- 月末集計を楽にしたい
- 予約ミスを減らしたい
- スタッフの稼働状況を見たい

### スタッフ

施術を担当する現場スタッフ。
日々の予約確認、顧客情報の確認、来店時のメモ更新を行う。

主な関心ごと：

- 今日の予約をすぐ確認したい
- 顧客情報を素早く探したい
- 顧客ごとの注意事項を確認したい
- 予約変更やキャンセルを簡単に処理したい

## 3. 現在の業務運用

### 予約管理

予約は電話、LINE、Instagram DM、店頭で受け付けている。
受け付けた予約は紙の予約表またはExcelに転記している。

### 顧客管理

顧客情報はExcelで管理している。
来店履歴やメモはスタッフごとに記録方法が異なる。

### 売上管理

日々の売上は手作業で集計している。
月末にExcelへまとめて入力している。

## 4. 困っていること

- 予約の転記漏れがある
- 同じ時間帯に予約が重なりそうになることがある
- 顧客情報を探すのに時間がかかる
- 顧客メモがスタッフ間で共有されにくい
- 売上集計に時間がかかる
- スタッフ別売上をすぐ確認できない

## 5. システムに期待すること

- 予約を一元管理したい
- 顧客情報を簡単に検索したい
- 顧客ごとの来店履歴を確認したい
- 予約重複を防ぎたい
- 会計後の売上を自動で集計したい
- 日次・月次の売上を画面で見たい
